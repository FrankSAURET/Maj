Aide – Gestionnaire d’extensions Inkscape

Configurer vos extensions personnalisées

  • Placez un fichier Info.json à la racine de chaque extension.
  • Le champ type doit valoir InkScape extension.
  • Indiquez la version, la liste des fichiers et le dossier d’installation par défaut.
  • Ajoutez vos dépôts dans l’onglet Paramètres.

Exemple de Info.json

{
  "type": "InkScape extension",
  "version": "1.0.0",
  "files": ["MonExtension.inx", "MonExtension.py"],
  "default_install_dir": "MonExtension"
}

Pour toute question ou contribution, consultez le README ou le dépôt GitHub.
